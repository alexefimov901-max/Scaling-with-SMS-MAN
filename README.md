## 1. Intro — Scaling with SMS-MAN

**Scaling with SMS-MAN** is about using virtual number infrastructure to handle large volumes of SMS verification.

When you move from a few accounts to hundreds or thousands, manual workflows stop working. You need automation, API access, and predictable limits.

With **Scaling with SMS-MAN**, the focus is on speed, stability, and the ability to run operations at scale without using personal numbers.

## 3. How Scaling with SMS-MAN works

**Scaling with SMS-MAN** works through API integration:

1. Connect to SMS-MAN API  
2. Send request for a number (country + service)  
3. Receive number via API  
4. Use number in your system  
5. Poll or receive SMS via webhook  

Typical flow:

- Request → Assign number → Wait for SMS → Confirm → Complete  

This removes manual steps and allows parallel processing.
